Web design : HTML & CSS

The goal was to create a user friendly website for asmall restaurant that want to start offering online ordering and takeaways services to the local community. They needed a basic website with an online menu to achieve this. In this course assignment we were tasked with the design, implementation and deployment of the said website. The website should function on various electronic devices (especially mobile phones) and be fully responsive. We are asked to ensure the following are included in the design:
• The website is visually inviting and capture the atmosphere of the restaurant.
• The website should also have at least Menu and Contact pages whereby the menu should be the focus and contact should be easily accessible.
• The website should be coded from scratch using HTML and CSS and deployed online
